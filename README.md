# SVG-Logo-Maker

## Description 
Create a svg logo with user input from question prompts.

## Setup Instructions
Right click on index.js and click on 'Open in integrated Terminal' type 'npm run test' to initiate a test run. Test suites and Tests should all pass.

## Usage Instructions
Right click on index.js and click on 'Open in integrated Terminal' type 'node index.js' to initiate promnpts. User should input answers to prompts and an SVG logo will generate under the title 'logo.svg'. Right click on 'logo.svg' and click 'Open with Live Server' and an svg logo generated by user inputs will display in live server.
